https://github.com/xksoft/v3/releases/download/%E4%B8%BB%E7%A8%8B%E5%BA%8F/Release.zip

## 老数据转移
老站点数据可直接复制到目录下的“系统数据//Main”中，复制完成启动后需要修改站点和任务信息，重新设置采集和发布模块。老模块可使用导入导出的方式添加到新版本中
